working remote via hdhomerun

Heh, I had an inspiration this morning:  what if I can get the IR receiver on my HDHomeRun to act as the controller for my MythTV frontend, and bypass the fact that the USB ports on the mobo are dead?

The instructions are here.  But I didn't know what remote to use.  Then I saw the Apple Remote that came with my macbook sitting on the shelf.  I tried a couple of LazyNet configuration files but they didn't work - even the one that is apparently the identical model.

So fire up irrecord a couple of times, read the instructions the last time, and bang, it starts to narrow down which remote I'm using.  Cool!

The config file I generated seems more verbose than the ones I saw provided by LIRC, but there's probably a reduction phase I'm not going to bother with.

Course, now that I know how easy it is, I should be able to use the programmable remote that has more than 6 buttons.

Now I just need to link LIRC up with MythTV via the lircrc file...
