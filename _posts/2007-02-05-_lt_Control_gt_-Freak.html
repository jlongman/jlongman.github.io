<Control>-Freak

Remember my concerns about being able to get the or key in the 'right' place?
Well, while looking for the setting to make the Laptop/Function-keys (things like brightness, volume, etc) to default to Function-key behaviour, there it was, a little button that said, 'Modifier Keys...'. And it does exactly what I was looking for, map or to that key right by my pinky finger. And SO EASY too!

My next desire is a script that will change it back and forth, and then Quicksilver access to that script. Then I'll be able to easily switch from code-mode to node-mode (the one where I sit on the couch).

BTW, speaking of control-freak, if you're not using Quicksilver you're a moron. Quicksilver is like pie-menus for your keyboard: a UI that works for beginners but scales effortlessly for advanced users. It takes the search behaviour of Spotlight and applies the same search intelligence to data AND actions.

It doesn't replace Spotlight - it doesn't seem capable of searching my file system as efficiently as Spotlight, but I suspect that Spotlight is doing a lot of heavy-lifting behind the scenes, and one app doing that is enough of a load on my beasty. Still I'm curious if they leverage Spotlight's DB.

I think there's a way to take any file from the Finder and have Quicksilver act on it, in which case the two pipeline into each other quite well.   I know you can walk the path using Quicksilver and then execute actions on things in the path...
