Linux Wifi SO Easy!

I can't believe it. The hardest thing about configuring the wifi was finding the UI to do it. It helps I'm using old hardware, but WOW!!! It was SO much easier than past times I've tried to do it.

Surprisingly, just getting the icons to work in Gnome was all screwed up. It worked the first time, but after that nada. Creating a new account managed to get me around the problem.

I suspect it had something to do with my key repeat configuration failing. My theory is that somehow the schema for gconf2 (Gnome Configuration?) was in the wrong place. And as the schema is supposed to tell it what type of parameter the saved configuration is, while passing data to the process which applies the configuration, the process which applies the configuration was complaining that the wrong type was passed.

(What interesting word usements I structure.)

I followed the Linux battery mini-howto and that took me to about 1am last night. The only thing REALLY left was the fact that I couldn't control the brightness on my machine. Ends up I had configurations in the wrong file. Now I just hav e to figure out how to boot up like that.
