Freemind viewer in Confluence 4.2

(me: Oh hullo blog, where you been? blog: Sneaking!)

Finally Confluence upgraded to a version less than 4 years old, but it broke my embedded freemind viewer. Freemind is a free as in speech and beer mindmap tool. Confluence is a wiki, and it's nice to view the mindmaps in the wiki.

  • Add a user macro for Freemind - match the options below
  • Get swfobject.js (or flashobject.js) and attach it to a page - note the URL location of the attachment - substitute it below
  • Get visorFreemind.swf and attach it to a page - note the URL location of the attachment - substitute it below

You can get the javascript and flash objects by following these instructions.

## Macro title: Freemind MindMap ## Macro has a body: N ## Body processing: No macro body ## Output: HTML ## ## Developed by: xxx ## Date created: xxx ## Installed by: xx ## Set the file ## @param file:title=Mindmap file|type=string|required=true|desc=Write the attachment name (e.g. mine.mm) ## Set the width for the player ## @param width:title=Viewer width|type=string|desc=Enter the width of the video in pixel (e.g. 640px)|required=true|multiple=false ## Set the height of the player ## @param height:title=Viewer height|type=string|desc=Enter the height of the video in pixel (e.g. 480px)|required=true|multiple=false
