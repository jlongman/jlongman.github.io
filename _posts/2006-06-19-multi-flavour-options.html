multi-flavour options

To dual-boot (er, triple?) or to virtual machine?

Neither one is perfect - for bootcamp, triple-booting is slightly fragile: there's a max number of partitions that are supported on your drive, meaning you can't create a linux swap partition, and if you're not careful you can overwrite the boot sector on the relevant drive. They don't suggest you use NTFS.

For Parallels, (i.e. virtual machines) you're running a virtual machine which has some overhead, meaning you're not really getting full speed. You need more memory. There were some reports its not really stable yet. (Those could be out of date though).

And what am I trying to achieve by doing this? Didn't I go mac because I wanted to have a system that just worked? At this point, as far as I know, all my tools run on OSX.

There's just a few small things bother me. One mouse button (minor). Fixed font sizes and screen space wastage (Maybe there's something that fixes that?). Will I be able to remap my control key to the right place? (Or maybe the Command key?) (Not to mention the multi-platform control vs. command key schizophrenia)

The longer I wait, the more work will be required if I need to format or re-partition.
