Move block keybindings

Brilliant demonstration here to map a keybinding that will move an entire line or selected block up and down in a document (see the comments more specifically). This is a feature that's really handing in coding, and I REALLY miss it from IntelliJ's editor. It seems like a lot of times I'm just trying to restore the functionality of IntelliJ (without the bloat and crashes that their support denied).

Bit of a link dump:

  • Source material on keybindings in OSX
  • Same thing in User Scripts
  • Key Bindings Editor is a user friendly way to acheive this

And a caveat that if you have custom key bindings in xcode then it changes the rules a bit, and you need to look in the plist file. Details are buried in the Completion Dictionary site, which might satisfy another need (automagic macros for log messages, for example).

Another caveat if not using KeyBindingsEditor and cutting and pasting from the site is be careful about smart-quotes when pasting from the web!
