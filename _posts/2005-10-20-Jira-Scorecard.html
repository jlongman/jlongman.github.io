Jira Scorecard

Here's a little tool for Jira, the bug tracking tool we use at work, that can be fun.

Create a filter for Jira, I called mine 'Scorecard':

  1. Project: (er, well, your project here)
  2. Issue Type: Defect, Improvement, New Feature
  3. Status: Closed
  4. Sorted by: Key descending

Then create a Two-Dimensional Filter Portlet, set up like this:

  1. Filter: Scorecard
  2. X Axis: Fix For Versions (non-archived)
  3. Y Axis: Assignee
  4. Show Totals: Yes

And this is what pops out:

This allows a little competitive spirit in fixing bugs. Of course, it doesn't address the problem of who's writing the bugs. And it definitely would be nice if there's a scaling for the priority. I'm basically in third position, but Mr. #1 managed to fix hundreds of typos, all Priority 5 (lowest). (Smart player!)
